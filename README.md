## Running the project
Clone this repository
  ### Backend
  1. Build the solution using [Visual Studio](https://visualstudio.microsoft.com/downloads/).
  2. Run the project. 
  ### Frontend
  
  In `/ChatHubUi/` folder run:
  ```
  npm i
  npm start
  ```
